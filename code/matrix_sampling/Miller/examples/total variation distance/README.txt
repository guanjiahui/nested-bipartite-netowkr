
Estimating the total variation distance between an approximate sampler and the uniform distribution.


===============================================================================
CONTENTS
===============================================================================

total_variation.m
This is the main script.

BernoulliMarginsRnd.m
canonical.m
These files implement the approximate sampler described in:
    M. T. Harrison and J. W. Miller (2013).
    Importance sampling for weighted binary random matrices with specified margins.
    arXiv:1301.3928 [stat.CO].

sample_Q.m
compute_Q.m
Interface for approximate uniform sampling from this approximation, and computing probabilities under it.
